# docking_tools

Local one-step AutoDock-GPU + gnina docking workflow.

Start here: [INSTALL_AND_RUN.md](INSTALL_AND_RUN.md).

Pipeline details: [README_ADGPU_GNINA_PIPELINE.md](README_ADGPU_GNINA_PIPELINE.md).

Quick use after installing the local bash alias:

```bash
dock box
dock receptor.pdb ligand.mol2 box.txt
```

The repository tracks the Python/Bash workflow and small example inputs only. Local installed binaries, downloaded archives, logs, OMX state, and docking run outputs are intentionally ignored.
